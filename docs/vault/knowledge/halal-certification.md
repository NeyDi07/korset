## Определение халал в контексте пищевых продуктов

Халал (حلال) — «дозволенное». В контексте продуктов питания означает:

- Не содержит свинину и её производные
- Не содержит алкоголь (этанол) в значимых количествах
- Не содержит мясо животных, забитых не по исламским правилам
- Не содержит кровь и её производные
- Не содержит насекомых и их производные (кармин, шеллак)
- Не содержит хищных животных и птиц с когтями

Харам (حرام) — «запрещённое». Прямо противоположно халал.

Машбух (مشبوه) — «сомнительное». Когда происхождение ингредиента неизвестно и нельзя определить халал/харам.

## Категории ингредиентов по халал-статусу

### Автоматически ХАРАМ (без проверки)

- Свинина и все производные: свиной жир, желатин из свинины, лард
- Алкоголь: этанол (исключение — естественное ферментативное образование в малых количествах, см. ниже)
- Кровь и её производные
- Мясо животных, забитых не по правилам (без упоминания имени Аллаха, удушенное, забитое хищником)
- Насекомые: кошениль (E120), лаковый червец (E904)
- Хищные животные, рептилии, насекомые-вредители

### Автоматически ХАЛАЛ (без проверки)

- Растительные ингредиенты в чистом виде: фрукты, овощи, зерно, орехи, бобовые
- Минеральные ингредиенты: соль, сода, диоксид кремния
- Синтетические ингредиенты из нефтехимии: большинство красителей, консервантов
- Молочные продукты от халал-животных (корова, овца, коза)
- Мёд
- Рыба (все виды, по большинству мазхабов)

### СОМНИТЕЛЬНЫЕ (требуют проверки — МАШБУХ)

Каждый из этих ингредиентов МОЖЕТ быть халал, а МОЖЕТ быть харам — зависит от источника:

| Ингредиент                   | Халал если                                                    | Харам если                                    |
| ---------------------------- | ------------------------------------------------------------- | --------------------------------------------- |
| **Желатин**                  | Из халал-животного (корова, забитая по правилам)              | Из свинины или некертифицированного источника |
| **Глицерин (E422)**          | Растительный (пальмовый, кокосовый) или синтетический         | Из животного жира (особенно свиной)           |
| **Лецитин (E322)**           | Соевый, подсолнечный                                          | Из животного жира или яичный без сертификации |
| **Моно-/диглицериды (E471)** | Из растительных масел                                         | Из животного жира (часто свиной)              |
| **Жирные кислоты (E470a)**   | Из растительных масел                                         | Из животного жира                             |
| **Стеариновая кислота**      | Из растительных масел                                         | Из животного жира                             |
| **Инозинат натрия (E631)**   | Синтетический или микробиологический                          | Из животного мяса/рыбы                        |
| **Пчелиный воск (E901)**     | Большинство халал-органов: разрешён                           | Некоторые организации: не разрешён            |
| **Спирт как растворитель**   | Если остаточный этанол <0.1% и не используется как ингредиент | Если этанол добавлен как ингредиент или >0.1% |

## Правила принятия решения для Fit-Check Yellow level

### Шаг 1: Проверка сертификата

Если у продукта есть официальный халал-сертификат → **статус = ХАЛАЛ**, не нужно проверять состав дальше. Сертификат признаётся если выдан:

- Казахстанским Исламским Центром
- JAKIM (Малайзия)
- MUI (Индонезия)
- HFCE (Европа)
- Halal Certification Services (Россия)
- Любым органом, признаваемым当地 исламским управлением

**Реализация:** поле `halal_status = 'yes'` в `global_products` таблице означает, что сертификат подтверждён.

### Шаг 2: Проверка состава (если сертификата нет)

Если `halal_status = 'unknown'` или не указан:

1. **Сканирование состава на харам-ингредиенты:**
   - Свиной жир, lard, pork fat, свинина
   - Алкоголь, ethanol, ethyl alcohol, спирт этиловый (исключение: натуральная ферментация <0.1%)
   - Желатин (gelatin) — без уточнения «растительный» или «халяль» → машбух
   - Кармин (E120, cochineal) → харам
   - Шеллак (E904, shellac) → харам

2. **Сканирование состава на машбух-ингредиенты:**
   - E471, E472a-e, E470a, E422, E322 — если нет пометки «растительный» → машбух
   - E481-483 — стеароиллактаты → машбух
   - E631 — если нет пометки «синтетический» → машбух

3. **Определение статуса:**
   - Найден харам → **статус = НЕ ХАЛАЛ** (danger / warning)
   - Найден машбух, нет харам → **статус = СОМНИТЕЛЬНЫЙ** (caution) — пользователь решает сам
   - Ни харам, ни машбух → **статус = ВЕРОЯТНО ХАЛАЛ** (pass с оговоркой)

### Шаг 3: AI-анализ (Yellow level — LLM-агент)

Для сомнительных Е-добавок, где детерминированный код не может дать однозначный ответ:

- LLM анализирует контекст: остальные ингредиенты, бренд, страну производства
- LLM опирается на RAG-базу знаний (vault/knowledge/e-additives.md) для оценки каждой добавки
- LLM НЕ заменяет детерминированные проверки — дополняет их

**Критически:** LLM-агент НЕ используется для Red level (аллергены, диабет) — там только детерминированный код.

## Спорные вопросы и нюансы

### Этанол в продуктах

- **Консенсус:** Продукты с добавленным алкоголь-ингредиентом = харам
- **Спор:** Естественно ферментированные продукты (кефир, квас, фруктовые соки) с <0.5% этанола
  - Большинство учёных: халяль (естественное образование, не добавленный)
  - Консервативная позиция: <0.1% для абсолютной уверенности
- **Реализация для Körset:** Если в составе указан «алкоголь», «спирт», «ethanol» как ингредиент → харам. Если естественная ферментация — предупреждение `caution`

### Уксус

- Спиртовой уксус (получен из этанола, но полностью ферментирован до уксусной кислоты) — **халяль** по большинству мнений
- Реализация: уксус (E260) = халяль

### Желатин

- Самый спорный ингредиент в пищевой промышленности
- Если указано «желатин» без уточнения → **машбух** (сомнительно)
- Если «растительный желатин» или «желатин халяль» → **халяль**
- В Казахстане ~70% желатина в продуктах — свиной (по оценкам пищевых экспертов)
- Реализация: «желатин» без уточнения → caution (сомнительно)

### Ферментация с животными средами

- Некоторые ингредиенты (E415 ксантан, E270 молочная кислота) производятся ферментацией
- Если среда ферментации содержит животные компоненты → продукт может быть нехаляль
- На практике: большинство крупных производителей используют растительные среды
- Реализация: проверять наличие халал-сертификата на продукт

## Казахстанский контекст

### Законодательство

- Казахстан — светское государство, нет обязательной халал-маркировки
- Добровольная халал-сертификация через Казахстанский Исламский Центр
- ТР ТС 022/2011 не требует указания халал-статуса

### Рынок

- ~70% населения Казахстана — мусульмане (потенциальная аудитория халал-проверки)
- Халал-сегмент растёт на ~15% в год
- Основные сети с халал-полками: Magnum, Small, Green Market
- Местные халал-бренды: Рамазан, Бахетле, Халяль-Продукт

### Реализация для Körset

- Халал-проверка = одна из ключевых фич для B2B2C модели
- Магазин с халал-полкой = дополнительная причина купить подписку Körset
- Fit-Check Yellow level = точка дифференциации от конкурентов
