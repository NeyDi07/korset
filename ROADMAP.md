# План развития Körset (ROADMAP)

Этот файл содержит список важных бизнес-фич и идей, которые необходимо реализовать для выхода приложения на серьезный уровень.

## Приоритет 1 (Юридическая безопасность и Доверие)
- [ ] **Дисклеймер ИИ и Аллергенов:** Обязательная плашка (Disclaimer), предупреждающая пользователя, что ИИ может ошибаться, и приложение носит рекомендательный характер. Решения о покупке (особенно при сильных аллергиях) остаются за пользователем. Нужно проработать формулировки для защиты от юридических претензий.

## Приоритет 2 (Польза для покупателей)
- [ ] **Цены:** Отображение актуальных цен на товары, так как в магазинах часто отсутствуют ценники. Это одна из главных "киллер-фич" приложения для пользователей.
- [ ] **Срок годности:** Раздел, где пользователь может сохранить купленный товар со сроком годности, чтобы 앱 (приложение) напоминало, когда срок подходит к концу (push-уведомления или индикаторы в профиле).
- [ ] **Карта магазинов:** Интерактивная карта, показывающая, какие магазины уже поддерживают систему Körset, чтобы пользователи знали, куда идти.
- [ ] **Истории / Советы дня:** Формат "Stories" (как в Instagram/Kaspi) на главном экране. Интересные факты о питании, полезные советы, актуальные новости (влияет на удержание пользователей).

## Приоритет 3 (Улучшение UX / UI)
- [ ] **Сканер:** Увеличить область (окно) распознавания штрих-кода, чтобы сканирование срабатывало быстрее и легче.
- [ ] **Фото из галереи:** Разрешить пользователю выбрать фотографию штрих-кода из галереи телефона, если камера не работает или фото было сделано ранее.
- [ ] **История сканов:** Улучшить/расширить экран истории (Добавить фильтрацию, поиск, графики).
- [ ] **Повтор Онбординга:** Добавить кнопку в Профиле, чтобы пользователь мог заново посмотреть стартовые слайды (Onboarding), если он что-то пропустил.
